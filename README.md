# 💎 Expense Vault

A premium dark expense tracker built with **React** and **Supabase**.

## Features

- 🔐 User authentication (sign up / sign in) via Supabase
- ➕ Add expenses with name, amount, category, and date
- 🗂️ Categories: Dining, Travel, Bills, Other
- 📊 Summary cards — total spend, monthly spend, transaction count, top category
- 🗑️ Delete any expense entry
- 💾 All data persisted in Supabase (auto-creates table on first login)

## Tech Stack

- React (JSX)
- Supabase (Auth + PostgreSQL via MCP)
- Claude AI API (claude-sonnet-4-6)

## Design

Dark luxury aesthetic with gold accents — obsidian background, color-coded categories.

## Getting Started

1. Clone the repo
2. Open `src/ExpenseTracker.jsx` in your Claude.ai artifacts environment
3. Make sure Supabase MCP is connected in Claude
4. Run — auth and DB setup happen automatically
